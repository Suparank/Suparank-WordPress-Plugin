# Plugin Assets

This folder contains assets for the WordPress Plugin Directory.

## Required Files

For WordPress.org submission, add these files:

### Icons
- `icon-128x128.png` - Plugin icon (128x128px)
- `icon-256x256.png` - Plugin icon high-res (256x256px)

### Banners
- `banner-772x250.png` - Plugin banner (772x250px)
- `banner-1544x500.png` - Plugin banner high-res (1544x500px)

### Screenshots
- `screenshot-1.png` - Settings page
- `screenshot-2.png` - Connection test

## Design Guidelines

- Use Suparank brand colors
- Keep designs clean and professional
- See [WordPress Plugin Assets Guidelines](https://developer.wordpress.org/plugins/wordpress-org/plugin-assets/)
